## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e Github
- Figma

## 💻 Projeto

O DevLinks é um agregador de links para usar como cartão de visitas online.

- [Acesse o projeto finalizado, online](https://reuel02.github.io/dev-links/)


## 🎓 Aprendizado

Fiz esse projeto para consolidar meu conhecimento com tecnologias de front-end (HTML e CSS) e praticar a integração de interatividade usando JavaScript.
